================
Reporting issues
================

Use the GitHub issue tracker to report bugs, request improvements, or discuss
unexpected behavior. An informative issue report includes:

* the installed `siibra-python` version (``python -c "import siibra; print(siibra.__version__"``),
* the Python version (``python --version``),
* a minimal code example that reproduces the issue,
* the complete error message or traceback, if available.

Bug report: https://github.com/FZJ-INM1-BDA/siibra-python/issues/new?template=bug_report.md
Feature request: https://github.com/FZJ-INM1-BDA/siibra-python/issues/new?template=feature_request.md
Other: https://github.com/FZJ-INM1-BDA/siibra-python/issues/new?template=blank-issue.md

For questions related to EBRAINS services, account access, or infrastructure
availability, the EBRAINS support channels may be more appropriate than a
software issue: https://ebrains.eu/contact
